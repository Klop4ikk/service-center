# service-center
service-center1 - всё что есть по проекту

Ресурсы - данные которые использовались

Er-диаграмма - визуализация свзяей в пдф формате

Папка отчёты  - хранит все отчёты по всем заданиям

# Как всё запустить самому?
1) для начала нужно будет скачать всё что находится в папк service-center1.
2) После этого перейти в терминал в папке проекта и вставить pip install -r requirements.txt
3) Далее в терминале прописать python app.py и сайт должен автоматически запуститься
